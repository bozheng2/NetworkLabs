Welcome to the Dashboard

You are one step away from creating your first app!

Please use the tabs to navigate and select the various configurations available. Once you have finalized the onfiguraton, a config file will be generated upon submission. This will then be sent to the server and used to build the app.

You can configure each of these settings independently and submit them for the server to start processing them.
